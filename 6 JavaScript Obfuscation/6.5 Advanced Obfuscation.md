## More Obfuscation
Tools that can be used for obfuscating code are:
- [JSF](http://www.jsfuck.com)
- [JJ Encode](https://utf-8.jp/public/jjencode.html) or [AA Encode](https://utf-8.jp/public/aaencode.html)

Such obfuscators usually make code execution/compilation very slow, so it is not recommended to be used unless for an obvious reason, like bypassing web filters or restrictions.
